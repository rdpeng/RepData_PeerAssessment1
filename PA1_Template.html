Processing Data

Loop through DATA.NA and take the sum() of the total steps for each day. Since this process will be completed again later to group DATA by day as well we should create a function to do this for us.

This function groupBYday() takes two parameters and returns a dataframe. The first parameter, data, defaults to our dataset, df.raw. The second parameter, remove_NA, is set to FALSE. When set to TRUE, the function will first remove the NA values from df.raw, then group by day. 

groupBYday <- function(data = df.data, remove_NA = FALSE) {
  #  Initialize catch variable
  daily.steps <- c()

  ##  Get Unique Days from DATA$date
  days <- unique(data$date)

  ##  For loop through days
  for (i in 1:length(days)) {
      ##  Create Total Steps for day
      daily.steps[i] <- sum(data[data$date == days[i],1], na.rm = remove_NA)
  }
  # Create Data Frame
  df <- data.frame(days = days, steps = daily.steps)

  return(df)
}

#  GROUP BY INTERVAL
groupBYint <- function (data) {
    ##  Get Unique Intervals from df.int$interval
    int <- unique(data$interval)
    int.steps <- c()

    ##  For loop through days; Sum total Steps
    for (i in 1:length(int)) {
      ##  Create Average Steps for Intervals
      int.steps[i] <- mean(data[data$interval == int[i],1], na.rm = TRUE)
      }
      ##  Fill Data Frame
      df <- data.frame(interval = int, steps = int.steps)
      return(df)
}
